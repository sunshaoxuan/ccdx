## 关于我们

**CC点心** 初期通过本网站，主要为大家提供**手工饺子**的选购与配送（或自提，以实际上线规则为准）。我们坚持现包现做，面皮筋道、馅料饱满，从经典鲜肉到时令蔬菜、从家常口味到创意搭配，用一只只饺子传递家常味道。

馄饨、更多点心或堂食体验，若日后扩展，会在此陆续更新；**当前阶段，网站核心就是：把饺子卖好、卖清楚。**
