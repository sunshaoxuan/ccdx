# CC点心

**手工现做 · 馅料丰富 · 地道中餐**

---

## 关于我们

**CC点心** 是一家以传统中餐点心为特色的餐厅。我们专注**饺子**与**馄饨**，更以**多种馅料的手工饺子**为招牌——从经典鲜肉到时令蔬菜，从家常口味到创意搭配，每一只都现包现煮，力求面皮筋道、馅料饱满、汤鲜馅香。

在这里，你可以尝到馄饨的爽滑、饺子的扎实，以及师傅们日复一日对手工温度的坚持。

---

## 特色

- **手工制作**：现包现做，不将就工业速冻口感  
- **馅料多样**：多种馅料可选，满足不同喜好与季节  
- **中餐点心**：馄饨、饺子及配套小食，一桌暖胃又暖心
